# Task 2: Data Visualization and Storytelling

## Objective
Create visualizations that tell a clear business story using sales data. Tools used: **Power BI** .

## Dataset
- Dataset: `Superstore.csv`
- Fields used: Order Date, Region/State, Category, Sub-Category, Sales, Profit, Quantity, Segment.

## Visualizations Created
- KPI cards: Total Sales, Total Profit, Total Orders, Profit Margin.
- Line chart: Sales trend over time.
- Bar chart: Sales and Profit by Category.
- Bar/Tree map: Sales by Sub-Category (top products).
- Map: Sales by State/Region.
- Bar chart: Profit by Customer Segment.
- Slicers: Region, Category, Segment, Date.

## Insights / Story
- Identified top-performing categories and segments.
- Highlighted regions with strong vs weak performance.
- Detected potential loss-making categories/regions.
- Suggested focusing on profitable categories and reviewing discounts in low-profit areas.

## Deliverables
- Power BI dashboard (PBIX file).
- Exported PDF / screenshots of the dashboard.
- README explaining approach and insights.
